# @yugutou/eslint-config

## 在大佬 [@antfu/eslint-config](https://npmjs.com/package/@antfu/eslint-config) 的基础上加点自己喜好的配置


## Usage

### Install

```bash
pnpm add -D eslint @yugutou/eslint-config
```

### Config `.eslintrc`

```json
{
  "extends": "@yugutou"
}
```

### publish newversion

```
pnpm release x.x.x
```
